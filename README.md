# Research_on_media_niches

## Project Overview
This project is a data-driven analysis of YouTube media niches globally and  in Nigeria, designed to uncover which content categories attract the highest audience engagement, views, and retention.  
The goal of this project is to guide content creators, marketers, and media strategists in understanding the most profitable and attention-grabbing niches on the Nigerian YouTube scene.

This project uses the YouTube Data API v3 to collect, analyze, and visualize data across different media categories such as:
-  Music  
- Entertainment  
- Lifestyle  
- Education  
- Technology  
- News & Politics  
- Sports  

By exploring these niches, the research identifies **trends, audience behavior patterns, and opportunities for content development and brand growth

Objectives
1. Analyze and compare engagement metrics (views, likes, comments) across major media niches.  
2. Identify which niches show strong growth potential and **sustained audience retention.  
3. Provide data-backed recommendations for media investment and content strategy.  
4. Present clear visualizations to support decision-making for Media practicioners and stakeholders.
